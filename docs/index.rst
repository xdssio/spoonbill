Welcome to Spoonbill's documentation!
=====================================
What is Spoonbill? Inspired by `ibis <https://ibis-project.org/docs/3.2.0/>`_.
Spoonbill is a Python library that provides a lightweight, universal interface for Key-Values data stores. Write once, run anywhere.

For fast prototyping, testing, and simplification of data pipelines for scalable data flows.

Installation
~~~~~~~~~~
``pip install spoonbill-framework``


Features
~~~~~~~~~~
1. A unified interface for all key-value data stores.
2. A simple, intuitive API.
3. A lightweight, fast, and flexible library.
4. Extra features like Search, batch inserts and retrieval on (almost) all backends.
